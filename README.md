# Machine Learning Day 2 Projects

# Loan Approval Prediction & Student Pass/Fail Prediction using Logistic Regression

## 📌 Overview

This repository contains two beginner-friendly Machine Learning classification projects built using **Logistic Regression**, a supervised learning algorithm. These projects demonstrate the complete machine learning workflow, including data preprocessing, model training, evaluation, and prediction using Python and Scikit-learn.

---

# 📂 Projects Included

## 1️⃣ Loan Approval Prediction

### Description

This project predicts whether a loan application will be **Approved** or **Rejected** based on applicant details such as income, loan amount, education, credit history, and other relevant features.

### Features

* Dataset Exploration
* Data Cleaning
* Missing Value Handling
* Label Encoding
* Feature Selection
* Train-Test Split
* Logistic Regression Model
* Model Training
* Accuracy Score
* Confusion Matrix
* Classification Report
* Loan Approval Prediction
* Prediction Probability

---

## 2️⃣ Student Pass/Fail Prediction

### Description

This project predicts whether a student will **Pass** or **Fail** based on factors such as study hours, attendance, and other academic features using Logistic Regression.

### Features

* Dataset Loading
* Data Preprocessing
* Train-Test Split
* Logistic Regression Model
* Binary Classification
* Model Evaluation
* Accuracy Score
* Confusion Matrix
* Classification Report
* Student Pass/Fail Prediction
